知识图谱API说明 知识图谱API 实体查询 关系查询 属性查询

概述

知识图谱API是一组用于访问知识图谱数据的接口，可以实现对知识图谱中的实体、关系、属性等进行查询和操作。

使用方法

使用知识图谱API需要先获取API密钥，然后通过HTTP请求访问API接口。API接口返回的数据格式为JSON。

以下是一个查询实体信息的示例：

GET/api/entity?name=张三&apikey=yourapikey

其中，name参数表示要查询的实体名称，apikey参数表示API密钥。

API接口列表

接口名称 请求方式 参数 返回值
/api/entity GET name,apikey 实体信息
/api/relation GET subject,predicate,object,apikey 关系信息
/api/property GET entity,property,apikey 属性信息
/api/search GET keywords,apikey 搜索结果
/api/similar GET entity,apikey 相似实体
/api/path GET start,end,apikey 路径信息
/api/ontology GET entity,apikey 本体信息
接口名称 请求方式 参数 返回值
/api/user POST username,password 用户信息
/api/login POST username,password 登录状态
/api/logout POST apikey 注销状态
/api/search GET keywords,apikey 搜索结果
